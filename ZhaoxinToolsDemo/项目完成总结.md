## 🔗 架构说明

### 三层架构

```
┌─────────────────┐
│   Vue 3 前端    │  端口: 3003
│  (ZhaoxinDemo)  │  功能: 文件上传、模板选择、进度显示
└────────┬────────┘
         │ HTTP API
         ▼
┌─────────────────┐
│ Spring Boot后端 │  端口: 8091
│  (ZhaoxinDemo)  │  功能: API代理、文件处理
└────────┬────────┘
         │ HTTP API
         ▼
┌─────────────────┐
│   肇新 SDK      │  端口: 8080 (后端) / 3000 (前端)
│                 │  功能: 文档抽取、模板管理、结果展示
└─────────────────┘
```

### iframe嵌套方式

```
Demo前端页面
├── Extract.vue (主功能页)
├── ExtractResult.vue (结果页)
│    └── <iframe src="SDK前端/结果页" />
└── TemplateManage.vue (模板管理)
     └── <iframe src="SDK前端/模板管理" />
```

### iframe嵌套说明

**为什么使用iframe？**
- ✅ 完全复用SDK的UI，无需重复开发
- ✅ SDK的功能升级自动生效
- ✅ 保持UI一致性
- ✅ 开发效率高

**iframe的URL规则：**
```
结果页：{SDK前端}/rule-extract/result/{taskId}?embed=true
模板管理：{SDK前端}/rule-extract/template-manage?embed=true
```

**embed=true参数的作用：**
- 隐藏SDK前端的侧边栏
- 隐藏SDK前端的顶部导航
- 只显示核心内容区域
