## 📋 核心代码文件详解

### 🌟 后端核心：ZhaoxinApiClient.java

**功能**：封装对肇新SDK的所有API调用

**方法列表**：
- ✅ `submitExtractTask()` - 提交文档抽取任务
- ✅ `getExtractStatus()` - 查询任务状态
- ✅ `getExtractResult()` - 获取抽取结果
- ✅ `cancelExtractTask()` - 取消任务
- ✅ `getExtractTasks()` - 获取任务列表
- ✅ `getPageImage()` - 获取页面图片

**特点**：
- 完整的错误处理
- 详细的日志记录
- 支持multipart文件上传
- 自动清理临时文件

---

### 🌟 后端核心：ExtractController.java

**功能**：提供RESTful API端点

**API端点**：
- ✅ `POST /api/extract/upload` - 上传并抽取
- ✅ `GET /api/extract/status/{taskId}` - 查询状态
- ✅ `GET /api/extract/result/{taskId}` - 获取结果
- ✅ `POST /api/extract/cancel/{taskId}` - 取消任务
- ✅ `GET /api/extract/tasks` - 任务列表
- ✅ `GET /api/extract/page-image/{taskId}/{pageNumber}` - 页面图片

## 📦 安装和运行

### 后端
```bash
cd backend
mvn clean package -DskipTests
java -jar target/zhaoxin-tools-demo-backend-1.0.0.jar
```

### 前端
```bash
cd frontend
npm install        # 首次运行
npm run dev        # 开发模式
npm run build      # 生产构建
```
