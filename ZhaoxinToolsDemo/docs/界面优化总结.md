### 2. ✅ 演示文档卡片 - 蓝色背景

**设计理念**：使用清新的蓝色渐变，突出演示文档的重要性

**视觉效果**：
```
┌──────────────────────────────┐
│ 📄 文档名称                   │  ← 蓝色渐变背景
│    描述信息                   │  ← 浅蓝到白色渐变
│    [模板ID标签]               │
└──────────────────────────────┘
```

**样式特点**：
- **背景渐变**：`linear-gradient(135deg, #e8f4fd 0%, #f0f9ff 100%)`
- **边框颜色**：`#b3d8ff`（浅蓝色）
- **图标背景**：`#409EFF`（Element Plus 蓝色）
- **图标颜色**：`#ffffff`（白色）
- **hover 效果**：
  - 背景变深：`linear-gradient(135deg, #d9ecff 0%, #e8f4fd 100%)`
  - 阴影增强：`0 4px 12px rgba(64, 158, 255, 0.25)`
  - 轻微上移：`translateY(-2px)`

**修改文件**：
- `ZhaoxinToolsDemo/frontend/src/components/DemoLayout.vue`

**代码示例**：
```scss
.doc-item {
  background: linear-gradient(135deg, #e8f4fd 0%, #f0f9ff 100%);
  border: 1px solid #b3d8ff;
  
  &:hover {
    background: linear-gradient(135deg, #d9ecff 0%, #e8f4fd 100%);
    box-shadow: 0 4px 12px rgba(64, 158, 255, 0.25);
  }
  
  .doc-icon {
    background: #409EFF;  // 蓝色背景
    color: #ffffff;       // 白色图标
  }
}
```

### 4. ✅ 文档抽取 - 上传限制提示

**问题**：Demo 版不支持自定义上传，但没有明确提示

**解决方案**：在用户尝试上传文件时，弹出友好的提示对话框

**提示内容**：
```
标题：提示

本演示环境暂不支持自定义上传文档，
请使用左侧的演示文档进行体验。

如需使用自定义文档抽取功能，
请联系我们获取完整版系统。

[我知道了]
```

**交互逻辑**：
1. 用户点击上传区域或拖拽文件
2. 触发 `handleFileChange` 事件
3. 弹出 MessageBox 对话框显示提示
4. 阻止文件上传，清空文件列表
5. 用户点击"我知道了"关闭对话框

**修改文件**：
- `ZhaoxinToolsDemo/frontend/src/views/ExtractMain.vue`

**代码示例**：
```javascript
const handleFileChange = (file) => {
  // Demo 版本提示用户使用演示文档
  ElMessageBox.alert(
    '本演示环境暂不支持自定义上传文档，请使用左侧的演示文档进行体验。\n\n如需使用自定义文档抽取功能，请联系我们获取完整版系统。',
    '提示',
    {
      confirmButtonText: '我知道了',
      type: 'info',
      dangerouslyUseHTMLString: false
    }
  )
  // 不设置文件，保持空白状态
  selectedFile.value = null
  fileList.value = []
}

const beforeUpload = (file) => {
  // 阻止文件上传
  return false
}
```
