# 完整组件库跳转功能

## 功能说明

在 Demo 应用的顶部导航栏中添加了"完整组件库"链接，方便用户从演示环境跳转到完整的肇新合同组件库系统。

## 视觉效果

```
┌─────────────────────────────────────────────────────────────┐
│ 📋 肇新合同组件Demo                                          │
│                                                               │
│  [智能文档抽取] [智能文档比对] [智能合同合成] [🖥️ 完整组件库↗] │
└─────────────────────────────────────────────────────────────┘
```

## 设计特点

### 视觉区分
- **边框样式**：有淡灰色边框 `border: 1px solid #dcdfe6`
- **蓝色文字**：`color: #409EFF`，与其他菜单的灰色区分
- **外部链接图标**：右侧显示 `↗` 图标，表明会在新窗口打开
- **左侧间距**：`margin-left: 8px`，与其他菜单项保持距离

### 交互效果
- **hover 效果**：
  - 背景色：`#ecf5ff`（浅蓝色）
  - 边框色：`#409EFF`（深蓝色）
- **新窗口打开**：`target="_blank"`，不影响当前页面

### 图标使用
- **显示器图标**：`<Monitor />` - 代表完整系统
- **外部链接图标**：`<TopRight />` - 表示新窗口打开

## 配置说明

### 后端配置
**文件**：`ZhaoxinToolsDemo/backend/src/main/resources/application.yml`

```yaml
# 肇新 API 配置
zhaoxin:
  api:
    base-url: http://localhost:8080
    timeout: 120000
    connect-timeout: 10000
  frontend:
    url: http://localhost:3000  # ← 前端组件库地址
```

### 前端实现
**文件**：`ZhaoxinToolsDemo/frontend/src/components/DemoLayout.vue`

```vue
<a 
  href="http://localhost:3000" 
  target="_blank"
  class="nav-item nav-external"
>
  <el-icon><Monitor /></el-icon>
  <span>完整组件库</span>
  <el-icon class="external-icon"><TopRight /></el-icon>
</a>
```

- 查看所有历史记录
- 配置高级选项
- 访问 API 文档

### 3. 系统对比
用户可以对比 Demo 版和完整版的功能差异。

## 修改建议

### 修改跳转地址
如果完整组件库部署在其他地址，修改配置文件：

```yaml
zhaoxin:
  frontend:
    url: https://your-domain.com  # 修改为实际地址
```

然后修改前端代码：

```vue
<a 
  href="https://your-domain.com"  <!-- 修改为实际地址 -->
  target="_blank"
  class="nav-item nav-external"
>
```

### 修改显示文字
可以根据需要修改文字：
- "完整组件库"
- "查看完整功能"
- "完整版系统"
- "进入主系统"
- "更多功能"

### 修改图标
可以使用其他 Element Plus 图标：
- `<Platform />` - 平台图标
- `<Promotion />` - 提升图标
- `<Link />` - 链接图标
- `<Setting />` - 设置图标
