```bash

```

### 步骤2：创建新业务代码

创建以下新文件：

1. **后端服务类**
   - `ZhaoxinApiClient.java` - SDK API客户端

2. **后端控制器**
   - `ExtractController.java` - 文档抽取控制器

3. **前端项目**
   - 完整的 Vue 3 项目结构
   - 侧边栏导航组件
   - 智能文档抽取页面

### 步骤3：测试

1. 启动肇新SDK服务（端口 8080）
2. 启动Demo后端（端口 8091）
3. 启动Demo前端（端口 3003）
4. 访问 http://localhost:3003
