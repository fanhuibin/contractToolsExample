#### 卡片阴影（参考 Ant Design）
```scss
box-shadow: 
  0 1px 2px rgba(0, 0, 0, 0.03), 
  0 1px 6px -1px rgba(0, 0, 0, 0.02), 
  0 2px 4px rgba(0, 0, 0, 0.02);

  box-shadow: 
    0 1px 2px rgba(0, 0, 0, 0.05), 
    0 4px 12px -2px rgba(0, 0, 0, 0.03), 
    0 8px 16px rgba(0, 0, 0, 0.03);
```

#### 圆角
- 主卡片：`8px` （Ant Design 标准）
- 按钮/输入框：`6px`

#### 配色方案
- **背景色**：`#f5f7fa` （页面背景）
- **卡片背景**：`#ffffff`
- **区域背景**：`#fafafa` （步骤指示器等）
- **边框色**：`#f0f0f0`

#### 间距（参考 Ant Design 8px 栅格）
- 外边距：`24px` (3x8)
- 内边距：`32px` (4x8), `24px` (3x8)
- 元素间距：`32px` (4x8), `16px` (2x8)

### 4. 问题与解决

#### 问题 1：布局变成纵向
**原因**：`max-width: 1200px` 太窄，触发了响应式样式

**解决**：
```scss
.main-card {
  max-width: 1400px; // 增加到 1400px
}
```

#### 问题 2：样式嵌套错误
**原因**：`.main-card` 内部又嵌套了 `.main-card`

**解决**：删除嵌套的 `.main-card`，所有子元素直接作为第一层 `.main-card` 的子级

## 如果布局仍然有问题

### 快速修复方案

如果横向布局还是显示为纵向，请检查浏览器窗口宽度：
- **最小宽度要求**：`1280px`
- **推荐宽度**：`1440px` 或以上

如果窗口宽度足够但仍然纵向排列，请：

1. **清除浏览器缓存**
   ```
   Chrome: Ctrl + Shift + Delete
   选择"缓存的图片和文件"
   ```

2. **强制刷新页面**
   ```
   Ctrl + F5 (Windows)
   Cmd + Shift + R (Mac)
   ```

3. **重启前端开发服务器**
   ```bash
   # 停止当前服务 (Ctrl+C)
   # 重新启动
   cd ZhaoxinToolsDemo/frontend
   npm run dev
   ```
