# 项目架构设计文档

## 整体架构

### 技术架构
- **前端**: Vue 3 + TypeScript + Vite + Element Plus
- **后端**: Spring Boot 2.7.x + MyBatis Plus + MySQL
- **文档处理**: OnlyOffice Document Server
- **文件存储**: 本地文件系统

### 系统模块
1. **用户管理模块** - 用户认证、权限管理
2. **合同管理模块** - 合同CRUD、状态管理
3. **模板管理模块** - 合同模板设计、管理
4. **文档处理模块** - OnlyOffice集成、文档编辑
5. **合同比对模块** - 合同版本比对、差异分析
6. **文档合成模块** - 多文档合并、格式转换

## 数据库设计

### 核心表结构
- `sys_user` - 用户表
- `contract_info` - 合同信息表
- `contract_template` - 合同模板表
- `contract_version` - 合同版本表
- `document_file` - 文档文件表

## API设计规范

### RESTful API
- GET /api/contracts - 获取合同列表
- POST /api/contracts - 创建合同
- PUT /api/contracts/{id} - 更新合同
- DELETE /api/contracts/{id} - 删除合同

### 响应格式
```json
{
  "code": 200,
  "message": "success",
  "data": {}
}
```
