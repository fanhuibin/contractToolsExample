### 3. 复用配色

#### 方式1：直接复制颜色值

```css
.my-button {
  background: #1890ff;
}
.my-button:hover {
  background: #40a9ff;
}
```

#### 方式2：使用CSS变量

```css
/* 定义变量 */
:root {
  --primary: #1890ff;
  --primary-hover: #40a9ff;
}

/* 使用变量 */
.my-button {
  background: var(--primary);
}
.my-button:hover {
  background: var(--primary-hover);
}
```
