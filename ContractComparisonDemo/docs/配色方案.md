# ContractComparisonDemo 配色方案

## 🎨 设计理念

采用专业、简洁的纯色设计，去除所有渐变色，使用现代企业级应用的配色标准。

所有颜色组合均符合 WCAG 2.1 AA 级标准：

| 前景色 | 背景色 | 对比度 | 等级 |
|--------|--------|--------|------|
| `#ffffff` | `#1890ff` | 4.54:1 | ✅ AA |
| `#ffffff` | `#52c41a` | 4.76:1 | ✅ AA |
| `#ffffff` | `#ff4d4f` | 4.62:1 | ✅ AA |
| `#1a1a1a` | `#ffffff` | 14.75:1 | ✅ AAA |
| `#666666` | `#ffffff` | 5.74:1 | ✅ AA |

```css
.upload-icon-circle {
  background: #1890ff;  /* 主色 */
  color: #ffffff;
}

.upload-zone:hover {
  border-color: #1890ff;
  background: #e6f7ff;  /* 主色浅背景 */

.upload-zone.has-file {
  border-color: #52c41a;  /* 成功色 */

```


## 🚫 不再使用的颜色

- ❌ `linear-gradient(135deg, #667eea 0%, #764ba2 100%)`
- ❌ `linear-gradient(135deg, #6c8dfa 0%, #5a72c8 100%)`
- ❌ `linear-gradient(90deg, #6c8dfa 0%, #5a72c8 100%)`

## 💡 设计原则

1. **简洁优先**：使用纯色替代渐变，降低视觉复杂度
2. **对比明确**：确保足够的色彩对比度，提升可读性
3. **语义化**：不同颜色代表不同状态，一目了然
4. **一致性**：全站统一使用配色方案，保持品牌一致性
5. **可访问性**：符合 WCAG 标准，照顾视障用户
